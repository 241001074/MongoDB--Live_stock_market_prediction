# MongoDB--Live_stock_market_prediction
This repo contains the mongoDB queries for live_stock_market_prediction

Aim:
To create a Live Stock Market Prediction system using MongoDB to perform database creation, insertion,
CRUD operations, validation, and verification of stock data.

Steps:

1. Open mongosh and create database using
   use stockDB

2. Create collection
   db.createCollection("stocks")

3. Add validation using collMod

4. Insert stock records using insertOne and insertMany

5. Read data using find()

6. Update data using updateOne / updateMany

7. Delete data using deleteOne / deleteMany

8. Verify data using countDocuments, sort, aggregate

9. Check final output using find().pretty()

Output:

 > Database created successfully
 > Collection created
 > Stock data inserted
 > CRUD operations executed
 > Validation applied
 > Aggregation and verification completed
 > Live Stock Market Prediction database works correctly
